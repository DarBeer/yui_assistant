# Yui Assistant 🤖

**Yui Assistant** — это современный ассистент, разработанный на платформе Node.js с использованием TypeScript. Проект имеет модульную структуру и полностью готов к компиляции и развертыванию в production-среде.

## ✨ Особенности

* **100% TypeScript**: Строгая типизация кода для повышения надежности и удобства поддержки.
* **Сборка через Yarn**: Быстрое и детерминированное управление зависимостями проекта.
* **Готовая конфигурация**: Настроенные файлы компиляции `tsconfig.json` и игнорирования `.gitignore`.
* **Разделение среды**: Архитектура проекта разделена на исходный код (`src/`) и финальную сборку (`dist/`).

## 🚀 Примеры кода для запуска

Вся основная логика приложения находится в папке `src/`. Ниже представлены стандартные скрипты для управления проектом, которые вы можете запустить из терминала:

```bash
# Запуск проекта в режиме разработки (с автоматическим перезапуском)
yarn dev

# Компиляция TypeScript кода в JavaScript
yarn build

# Запуск скомпилированной production-версии проекта
yarn start
```

## 🛠 Пошаговая инструкция по развертыванию

Выполните следующие шаги для локального запуска проекта на вашем компьютере:

### Шаг 1: Подготовка окружения
Убедитесь, что на вашем компьютере установлены:
* **Node.js** (рекомендуется версия LTS)
* **Yarn** (актуальная версия)

### Шаг 2: Клонирование репозитория
Склонируйте данный репозиторий и перейдите в папку проекта:
```bash
git clone https://github.com
cd yui_assistant
```

### Шаг 3: Установка зависимостей
Установите все необходимые пакеты, указанные в `package.json`:
```bash
yarn install
```

### Шаг 4: Сборка проекта
Скомпилируйте исходный код из папки `src/` в готовую для запуска папку `dist/`:
```bash
yarn build
```

### Шаг 5: Запуск приложения
Запустите собранное приложение в production-режиме:
```bash
yarn start
```
