# Yui Assistant 🤖

🌐 **Языки:** [English](README.md) | [Русский](README_RU.md)

---

**Yui Assistant** — это умный Telegram-бот на базе искусственного интеллекта Google Gemini. Проект разработан на платформе Node.js с использованием TypeScript, обладает модульной архитектурой и готов к развертыванию как локально, так и на сервере.

## ✨ Особенности

* **Интеграция с Gemini API**: Использует продвинутые языковые модели от Google для генерации ответов и ведения диалога.
* **100% TypeScript**: Строгая типизация кода для повышения надежности, предсказуемости и удобства поддержки приложения.
* **Гибкая конфигурация**: Поддержка чтения переменных как из локального файла `.env`, так и напрямую из переменных среды ОС (Environment Variables).
* **Сборка через Yarn**: Быстрое и безопасное управление зависимостями проекта.
* **Разделение среды**: Архитектура проекта разделена на исходный код (`src/`) и финальную сборку (`dist/`).

## ⚙️ Настройка переменных среды

Для работы бота необходимо передать ключи авторизации. Приложение умеет считывать их двумя способами:
1. Из файла `.env` в корневой папке проекта (удобно для локальной разработки).
2. Из системных переменных окружения (подходит для Docker, Heroku, Cloud Run, Vercel и т.д.).

Создайте файл `.env` в корне проекта и добавьте следующие параметры:

```env
# Токен вашего Telegram-бота, полученный от @BotFather
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz

# API ключ от Google AI Studio для доступа к моделям Gemini
GEMINI_API_KEY=AIzaSyYourActualGeminiApiKeyHere
```

> ⚠️ **Важно:** Никогда не фиксируйте файл `.env` в системе контроля версий Git. Он уже добавлен в `.gitignore`.

## 🚀 Примеры кода для запуска

Управление жизненным циклом приложения осуществляется с помощью стандартных CLI-скриптов:

```bash
# Запуск проекта в режиме разработки (с автоперезапуском при изменении кода)
yarn dev

# Компиляция TypeScript кода в JavaScript (в папку dist/)
yarn build

# Запуск скомпилированной production-версии проекта
yarn start
```

## 🛠 Пошаговая инструкция по развертыванию

Выполните следующие шаги для локального или серверного запуска бота:

### Шаг 1: Подготовка окружения
Убедитесь, что в вашей системе установлены:
* **Node.js** (рекомендуется версия LTS)
* **Yarn** (актуальная версия)

### Шаг 2: Клонирование репозитория
Склонируйте репозиторий и перейдите в созданную директорию:
```bash
git clone https://github.com
cd yui_assistant
```

### Шаг 3: Установка зависимостей
Установите все необходимые Node.js пакеты, указанные в `package.json`:
```bash
yarn install
```

### Шаг 4: Конфигурация приложения
Создайте файл `.env` по примеру из раздела «Настройка переменных среды» и впишите туда свои реальные ключи. Если вы деплоите бота на хостинг, просто задайте аналогичные `ENV` переменные в панели управления хостинга.

### Шаг 5: Сборка проекта
Скомпилируйте исходный код TypeScript в JavaScript перед запуском:
```bash
yarn build
```

### Шаг 6: Запуск ассистента
Запустите собранное приложение в Production-режиме:
```bash
yarn start
```
